# Modern QuantumPost API Client

A lightweight, modern API client built with React, Vite, and Tailwind CSS, with Electron support for desktop use.

## Features

- **Multi-tab requests**: Open multiple endpoints simultaneously
- **BASE URL & environment support**: Easily switch between different environments
- **Request headers and JSON body editor**: Full control over your requests
- **Response viewer**: Prettified JSON, raw text, status code, and timing information
- **Collections & saved requests**: Organize your API requests
- **Dark/Light theme support**: Choose your preferred theme
- **Localhost API testing**: Perfect for local development with Electron desktop app
- **Lightweight and fast**: No unnecessary bloat

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone <repository-url>
cd modern-postman
```
